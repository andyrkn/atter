Abstract

Atter is a solution aimed at better coordinating any type of activity. An activity owner is able to track any type of check-in, legal ones or frauds, to grade atendees and to give them feedback per check-in.

Register page

The user will be promted for his data in orded to create a new account.

Login page

User data such as email and password will be required for login.

Home page

If you are logged in the home page will display two containers. The first one contains the list of user own activities and the second one contains a list of user followd activities. If you are not logged in, simple information about the application will be available

Create activity page

This is the page for creating a new activity. An user can choose multiple options for his newly created activity.

Follow activity page

This page asks for a simple follow code in order to be able to follow an existing activity

Dashboard

Here a owner can see the stats of his activity, legal checkins, frauds, grades, graphs and he can also import or export data to or from the application. A follower will simply see the data the onwer allows him to.

Profile page

As an user you can see your first name, last name and email address. He can modify the first two of them. Also he can require or revoke dropbox access.

Import page

If you own the activity then you will be promted to get the files from dropbox. The files that the user wants to import must be situated in "atter-resources" directory which is located in the dropbox home folder. Only files with the .json and .csv extension will be imported. When clicking on a file data will be automatically imported into the activity.
