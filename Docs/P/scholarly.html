Features

  • Creating different activities form an organiser point of view
  • Following different activities form an attendee point of view
  • Attend a activity when check-in is enabled
  • Enabling checking-in as an organiser
  • Viewing a history of check-ins per activity, including graphics and fraud attempts
  • As an organizer you can assign different types of grading based on own preference to any attendee
  • As an organizer you could import data from external sources
  • As an organizer or attendee, if allowed, you could export data as html or json fromat
  • As an organizer you could change the type of grading privacy
    • Private: Only the organizer is able to view the grades
    • Protected: Each attendee can view his own grades
    • Public: All data is available for the attendees following an event
